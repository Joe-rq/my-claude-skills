# Unicode and ASCII Characters

Use this file for character tables and visual glyphs.

## Box Drawing

```text
Unicode
H: ─  V: │
TL: ┌  TR: ┐
BL: └  BR: ┘
JT: ┬  JB: ┴
JL: ├  JR: ┤

ASCII
H: -  V: |
TL: +  TR: +
BL: +  BR: +
```

## Arrows

```text
Unicode
Up: ▲   Down: ▼
Left: ◄ Right: ►
UL: ◤  UR: ◥
LL: ◣  LR: ◢

ASCII
Up: ^   Down: v
Left: < Right: >
```

## Dashes

```text
Unicode
dashH: ╌
dashV: ┊

ASCII
dashH: .
dashV: :
```
