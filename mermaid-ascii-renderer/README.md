# mermaid-ascii-renderer

beautiful-mermaid ASCII 渲染系统技能。

## 文档结构

- **SKILL.md** - 技能主体，包含快速开始、API 参考
- **references/** - 详细参考文档
  - `core-systems.md` - 核心渲染系统（类型、画布、绘图、网格布局、A* 路由）
  - `diagrams.md` - 各图表类型的渲染实现（流程图、序列图、类图、ER 图）
  - `characters.md` - Unicode/ASCII 字符表
  - `troubleshooting.md` - 故障排查与性能优化
  - `resources.md` - 外部参考链接

## 相关项目

基于 [lukilabs/beautiful-mermaid](https://github.com/lukilabs/beautiful-mermaid) - 一个将 Mermaid 图表转换为精美 ASCII/Unicode 渲染的开源项目（MIT 协议）
