# Medical Triage Skill (智能医疗导诊)

智能医疗导诊助手，模拟资深分诊护士进行结构化问诊。

## 功能

- 🚨 **急症识别** — 优先排查危险信号，及时引导急诊
- 💬 **OPQRST 问诊** — 标准化问诊流程，全面收集症状信息
- 🏥 **科室匹配** — 基于症状智能推荐就诊科室
- 📋 **病历生成** — 输出标准化预问诊记录

## 安装

```bash
# 复制到 Claude Code skills 目录
cp -r medical-triage ~/.claude/skills/

# Windows
xcopy medical-triage %USERPROFILE%\.claude\skills\ /E /I
```

## 文件结构

```
medical-triage/

```

## 使用场景

| 场景 | 示例输入 |
|------|----------|
| 症状问诊 | "我最近老是头疼" |
| 急症识别 | "胸口很疼，喘不上气" |
| 科室咨询 | "皮肤痒应该挂什么科" |
| 问诊记录 | "帮我整理一下刚才的问诊信息" |
